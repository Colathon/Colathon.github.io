# Mip-Splatting: Alias-free 3D Gaussian Splatting —— Method 部分阅读报告

> 论文主题：分析原始 3DGS 的 **屏幕空间膨胀 (screen space dilation)** 在改变采样率（拉近 / 拉远相机）时会带来 **腐蚀 (erosion)**、**高频伪影** 与 **混叠 (aliasing)**，
> 并提出两个滤波器替换它：**3D 平滑滤波器 (3D Smoothing Filter)** 从表示层面把 3D 场景的频率限制在训练视角所决定的 Nyquist 上限之内；
> **2D Mip 滤波器 (2D Mip Filter)** 在渲染时用一个近似 box filter 的 2D 高斯替换膨胀操作。二者结合实现跨采样率的无混叠渲染。

---

## 0. 总览：Method 在解决什么

Mip-Splatting 要修的，是 3DGS 里一个平时"看不出问题、一变焦就露馅"的操作 —— **屏幕空间膨胀 (dilation)**。逻辑链是：

1. **§4 采样率敏感性**：先讲清楚 3DGS 为什么会有膨胀操作，以及这个操作把 3D 高斯的真实尺度"隐式地"和 **训练时的采样率** 绑死了。一旦渲染采样率偏离训练采样率（拉近 / 拉远），膨胀就会以物理上错误的方式表现，产生腐蚀、高频伪影（拉近）和膨胀伪影（拉远）。
2. **§5.1 3D 平滑滤波器**：从 **多视角频率上界** 出发，对每个高斯基元估计它在所有训练视角中能达到的 **最大采样率** $\hat{\nu}_k$，用一个 3D 高斯低通滤波器把该基元的最高频率限制在 $\hat{\nu}_k / 2$ 以内。这个滤波器 **烘焙进 3D 表示本身**，训练后固定不变。
3. **§5.2 2D Mip 滤波器**：用一个覆盖单像素的 2D 高斯滤波器（近似成像过程中的 box filter）替换原来的屏幕空间膨胀，在拉远 / 缩小采样率时抑制混叠。

一句话概括创新点：**膨胀是"事后打补丁"式地防止 2D 高斯退化成亚像素点，它没有物理依据、且隐式绑定训练采样率；Mip-Splatting 用有物理依据的采样理论 (Nyquist) 替换它，把"频率约束"这件事从渲染阶段前移到表示阶段。**

## 1. §4 采样率敏感性 (Sensitivity to Sampling Rate)

这一节是理解全文的钥匙，也是你当时卡住的地方。我把它拆成三步讲清楚：**膨胀是什么 → 膨胀干了什么坏事 → 为什么坏事表现为混叠 / 高频伪影。**

### 1.1 先回顾：膨胀 (Dilation) 是什么、为什么 3DGS 需要它

3DGS 渲染时，每个 3D 高斯先投影到屏幕空间得到一个 2D 高斯 $\mathcal{G}_k^{2D}$，再按深度顺序做 alpha blending：

$$
\mathbf{c}(\mathbf{x}) = \sum_{k=1}^{K} \mathbf{c}_k\, \alpha_k\, \mathcal{G}_k^{2D}(\mathbf{x}) \prod_{j=1}^{k-1}\left(1 - \alpha_j\, \mathcal{G}_j^{2D}(\mathbf{x})\right) \tag{4}
$$

问题在于：如果某个投影后的 2D 高斯 **在屏幕空间太小（小于一个像素）**，它就落在像素采样格点之间、几乎采不到，梯度也几乎为零，优化会不稳定甚至丢失这个基元。为避免这种 **退化 (degenerate)** 情况，3DGS 对投影后的 2D 高斯做 **膨胀**：

$$
\mathcal{G}_k^{2D}(\mathbf{x}) = e^{-\frac{1}{2}(\mathbf{x}-\mathbf{p}_k)^T (\Sigma_k^{2D} + s\mathbf{I})^{-1} (\mathbf{x}-\mathbf{p}_k)} \tag{5}
$$

这里 $\mathbf{I}$ 是 2D 单位矩阵，$s$ 是标量膨胀超参数（尺寸 $\approx 1$ 像素）。关键要看懂式 (5) 做了两件事：

- **协方差 $+ s\mathbf{I}$**：把 2D 高斯"撑大"，保证它在屏幕空间至少覆盖约 1 个像素，于是总能被采样到。这类似形态学 (morphology) 里的膨胀算子，故得名。
- **最大值不变**：注意归一化系数没有随之改变（式 5 前面没有像式 9、式 10 那样乘一个尺度修正因子）。也就是说膨胀 **只把高斯摊得更宽、峰值高度不变**，因此 **总能量（积分）被人为放大了**。这一点是后面"拉远变亮"伪影的根源。

### 1.2 关键：膨胀把"3D 尺度"和"训练采样率"绑死了

想象论文 Figure 1 的极简例子：一个物体、一个只有 5 个像素的图像传感器。优化的目标是"渲染出来的图像匹配训练图像"。这里有一个 **尺度歧义 (scale ambiguity)**：

- 一个 **尺寸正常** 的 3D 高斯，投影到屏幕后本来就 $\ge 1$ 像素，膨胀几乎不改变它。
- 一个 **尺寸极小（趋于 Dirac δ）** 的 3D 高斯 —— 本该投影成一个亚像素的点 —— 经过膨胀 $+ s\mathbf{I}$ 后，在屏幕上被 **强行撑大到约 1 像素**。于是它渲染出来的图像，和那个"尺寸正常"的高斯 **看起来几乎一样**。

优化器只看训练图像的光度损失，这两种解它 **无法区分**。而稠密化 / 优化过程天然倾向于用 **一大堆过小的高斯** 去拟合复杂场景（尤其像 Mip-NeRF 360 这种）。结果就是：**为了在训练采样率下匹配图像，模型系统性地低估了高斯的真实 3D 尺度** —— 因为膨胀会替它们"补上"缺失的屏幕尺寸。

> 核心一句话：**膨胀是在屏幕空间补偿的，补偿量 $s$ 固定约 1 像素；但"1 像素对应多大的世界空间"取决于采样率（相机远近 / 焦距）。训练时用这个补偿凑对了图像，就等于把高斯的真实尺度悄悄绑定在了"训练时的那个采样率"上。**

这就是为什么"渲染速率和训练速率相近时不出问题"（论文原话：*does not affect rendering at similar sampling rates*），但一旦变焦就出问题。

### 1.3 为什么表现为混叠 / 高频伪影 —— 拉近和拉远分开看

有了 1.2 的绑定关系，两种伪影就都能推出来了。**判据只有一个：那个被膨胀撑大的高斯，它填充的固定屏幕尺寸 $s$（约 1 像素）在新采样率下"对不对得上"新的像素大小。**

**情形 A：拉近 / 放大相机（Figure 1(d)，采样率升高）**

拉近后，同一个物体在屏幕上占的像素变多了，**每个像素对应的世界空间更小**（采样更密）。但那个在训练中被 **低估了真实尺度** 的高斯，其真实（未膨胀）尺寸本来就偏小；现在采样变密，膨胀那 $+ s\mathbf{I}$（约 1 旧像素）相对于新的更密采样已经不足以覆盖，高斯投影后 **变小**。于是：

- 渲染出的物体结构 **比真实更细 (thinner than they actually appear)** —— 这就是 **腐蚀 (erosion)**。论文举例：自行车轮辐会被渲染得比实际更细。
- 更要命的是，这些过小的、被低估尺度的高斯，其在屏幕上的能量分布 **尺寸小于两倍采样间隔**，违反 Nyquist（见 §5 的 $2\hat{T}$ 判据），采样时无法被正确重建 → **高频伪影 (high-frequency artifacts)**，视觉上就是闪烁 / 摩尔纹式的混叠。

**情形 B：拉远 / 缩小相机（Figure 1(c)，采样率降低）**

拉远后，物体在屏幕上占的像素变少，**每个像素对应的世界空间更大**。极端情况下，一个 3D 物体的投影 **比一个像素还小**。此时：

- 按物理，投影小于一个像素的物体，其辐射应该被 **像素面积平均掉、衰减变暗**（能量摊到整个像素里）。
- 但膨胀后的高斯 **峰值不衰减**（回忆 1.1：膨胀不改最大值、还人为放大了总能量），它把辐射以 **物理上错误的方式** 铺满这块像素，**累积了比实际到达该像素更多的光** → 画面 **异常变亮**、出现 **膨胀伪影 (dilation artifacts)**。论文同样以轮辐为例：拉远时轮辐会因这种错误的亮度扩散而严重劣化。

### 1.4 那为什么不干脆丢掉膨胀？

自然的想法：既然膨胀有害，训练时直接不用不就行了？论文给出两点，说明这条路走不通，从而 motivate 了 §5 的替换方案：

1. **优化困难**：对 Mip-NeRF 360 这类复杂场景，稠密化机制会造出海量小高斯，直接丢掉膨胀会导致优化不稳定，且这些小高斯 **超出 GPU 容量**。
2. **仍会混叠**：即便某个模型能在无膨胀下成功训练，**降低采样率（拉远）时依然会因缺乏抗锯齿而混叠** —— 因为膨胀（哪怕有害）好歹是唯一在管"频率"的操作，删了它连唯一的补丁也没了。

**结论**：膨胀不能简单删除，而要用 **有物理 / 采样理论依据的滤波器** 替换 —— 这正是 §5 做的事。

## 2. §5 Mip Gaussian Splatting

针对 §4 的诊断，论文对原始 3DGS 做两处修改：

- **§5.1 3D 平滑滤波器**：在 **表示层面**，把 3D 场景的频率限制在"训练图像所决定的最大采样率的一半"以内，消除 **拉近** 时的高频伪影。
- **§5.2 2D Mip 滤波器**：在 **渲染层面**，用一个近似成像 box filter 的 2D Mip 滤波器替换屏幕空间膨胀，缓解 **拉远** 时的混叠与膨胀问题。

论文脚注特别说明：这里的 *alias* 是个统称，涵盖前文所有伪影 —— 膨胀、腐蚀、过度平滑、高频伪影，以及混叠本身。

### 2.1 §5.1 3D 平滑滤波器 (3D Smoothing Filter)

**动机：** 多视角三维辐射场重建是经典的 **病态 (ill-posed)** 问题 —— 多个不同的 3D 重建可以产生相同的 2D 投影（1.2 的尺度歧义正是其一）。关键洞察是：**一个被重建的 3D 场景，其最高频率受训练视角所定义的采样率所限。** 沿用 Nyquist 定理，我们要在优化时就把 3D 表示的最大频率约束住。

#### 2.1.1 多视角频率上界 (Multiview Frequency Bounds)

多视角图像是连续 3D 场景的 2D 投影。离散的图像网格决定了我们从连续 3D 信号上 **采样点的位置**，这个采样率同时取决于 **图像分辨率、相机焦距、场景到相机的距离**。

对焦距为 $f$（像素单位）的图像，屏幕空间采样间隔为 1。把这个像素间隔 **反投影到 3D 世界空间**，在深度 $d$ 处得到世界空间采样间隔 $\hat{T}$，其倒数即采样频率 $\hat{\nu}$：

$$
\hat{T} = \frac{1}{\hat{\nu}} = \frac{d}{f} \tag{6}
$$

按 Nyquist 定理，以频率 $\hat{\nu}$ 采样，重建算法能还原的信号频率上限为 $\frac{\hat{\nu}}{2} = \frac{f}{2d}$。因此，**一个尺寸小于 $2\hat{T}$ 的基元会在 splatting 时产生混叠**，因为它的尺寸低于两倍采样间隔 —— 这正是 §4 中"过小高斯 → 混叠"的定量版本。

#### 2.1.2 每基元的最大采样率 $\hat{\nu}_k$

为简化，用基元中心 $\mathbf{p}_k$ 近似深度 $d$，并忽略遮挡。由于采样率与深度相关、且随相机不同而不同，对基元 $k$，取它在 **所有能看到它的相机中的最大采样率**：

$$
\hat{\nu}_k = \max\left(\left\{ \mathbb{1}_n(\mathbf{p}_k)\cdot \frac{f_n}{d_n} \right\}_{n=1}^{N}\right) \tag{7}
$$

其中 $N$ 是图像总数，$\mathbb{1}_n(\mathbf{p})$ 是 **可见性指示函数**：当高斯中心 $\mathbf{p}_k$ 落入第 $n$ 个相机的视锥内时为真。

> **为什么取 max（而不是 min）？** 直觉是：我们希望 **至少存在一个相机** 能够正确重建该基元。取最大采样率，等于以"看得最清楚（最近 / 焦距最大）的那个视角"为准来设定该基元允许的最高频率 —— 只要有一个视角能撑起这个频率，就不把它压得更低。Figure 3 用 5 个不同深度 / 焦距的相机演示：camera 3 决定了最小的 $\hat{T}$（即最大采样率 $\hat{\nu}$）。

#### 2.1.3 用 3D 高斯低通滤波器约束频率

对每个 3D 高斯基元 $\mathcal{G}_k$，在投影到屏幕之前，先与一个 3D 高斯低通滤波器 $\mathcal{G}_{\text{low}}$ 卷积：

$$
\mathcal{G}_k(\mathbf{x})_{\text{reg}} = (\mathcal{G}_k \otimes \mathcal{G}_{\text{low}})(\mathbf{x}) \tag{8}
$$

这一步 **高效** 的原因：两个高斯的卷积仍是高斯，且协方差相加（$\Sigma_1 + \Sigma_2$）。于是：

$$
\mathcal{G}_k(\mathbf{x})_{\text{reg}} = \sqrt{\frac{|\Sigma_k|}{\left|\Sigma_k + \frac{s}{\hat{\nu}_k^2}\mathbf{I}\right|}}\; e^{-\frac{1}{2}(\mathbf{x}-\mathbf{p}_k)^T \left(\Sigma_k + \frac{s}{\hat{\nu}_k^2}\mathbf{I}\right)^{-1} (\mathbf{x}-\mathbf{p}_k)} \tag{9}
$$

对比 §4 的膨胀式 (5)，请注意三处关键差异 —— 这也是 3D 平滑滤波器"对"而膨胀"错"的原因：

1. **滤波在 3D 世界空间做，不在屏幕空间做**：所以它是表示的内在属性，不依赖某次渲染的采样率。
2. **滤波尺度 $\frac{s}{\hat{\nu}_k^2}$ 是每基元自适应的**：因为 $\hat{\nu}_k$ 依赖该基元可见的训练视角。$s$ 是控制滤波器大小的标量超参。这保证了 **任意高斯的最高频率分量都不超过它在至少一个相机下最大采样率的一半**。
3. **有归一化前因子 $\sqrt{|\Sigma_k| / |\Sigma_k + \cdots|}$**：它 **修正了总能量**（膨胀式 5 缺的正是这一项），因此不会像膨胀那样人为放大亮度。

由于 $\mathcal{G}_{\text{low}}$ 成为 3D 表示的固有部分，**训练后保持不变**。它从根上消除了 §4 情形 A 里"拉近时过小高斯违反 Nyquist"的高频伪影。

### 2.2 §5.2 2D Mip 滤波器 (2D Mip Filter)

**动机：** 3D 平滑滤波器解决了拉近（升采样率）的高频问题，但在 **更低采样率**（拉远 / 相机远离）渲染时 **仍会混叠**（§4 情形 B）。为此，用 **2D Mip 滤波器** 替换 3DGS 的屏幕空间膨胀。

**物理依据：** 复刻真实成像过程 —— 打到相机传感器某像素上的光子，是在 **该像素面积上积分** 的。理想模型应在图像空间用 **2D box filter**；为效率，用 **2D 高斯滤波器** 近似之：

$$
\mathcal{G}_k^{2D}(\mathbf{x})_{\text{mip}} = \sqrt{\frac{|\Sigma_k^{2D}|}{|\Sigma_k^{2D} + s\mathbf{I}|}}\; e^{-\frac{1}{2}(\mathbf{x}-\mathbf{p}_k)^T (\Sigma_k^{2D} + s\mathbf{I})^{-1} (\mathbf{x}-\mathbf{p}_k)} \tag{10}
$$

其中 $s$ 取为 **恰好覆盖屏幕空间一个像素**。同样地，注意式 (10) 与膨胀式 (5) 的核心区别：**多了归一化前因子** $\sqrt{|\Sigma_k^{2D}| / |\Sigma_k^{2D} + s\mathbf{I}|}$。正是这一项修正了总能量：当投影小于一个像素时，2D Mip 滤波器会 **正确地衰减（变暗）** 该高斯，而不是像膨胀那样保持峰值不变、错误地累积过多光线。这直接消除了 §4 情形 B 的"拉远变亮 / 膨胀伪影"。

#### 2.2.1 与 EWA 滤波器的区别

Mip 滤波器与经典的 **EWA (Elliptical Weighted Average) 滤波器** 表面相似，但底层原理不同：

- **Mip 滤波器**：目标是复刻成像过程中的 **box filter**，追求对单个像素的 **精确近似**。
- **EWA 滤波器**：目标是限制渲染信号的 **带宽**，其滤波器大小是 **经验选取** 的。EWA 论文甚至主张用单位协方差矩阵，等效于在屏幕上占据一个 3×3 像素区域 —— 但这会导致 **过度平滑 (overly smooth)** 的结果。

也就是说，Mip 滤波器有明确的物理目标（单像素 box），而 EWA 的核大小是拍脑袋定的、且偏大，容易糊。

## 3. 小结：两个滤波器如何各司其职

| | 3DGS 膨胀 (dilation, 式 5) | 3D 平滑滤波器 (式 9) | 2D Mip 滤波器 (式 10) |
|---|---|---|---|
| 作用空间 | 屏幕空间 | **3D 世界空间** | 屏幕空间 |
| 尺度依据 | 固定 $s\approx 1$ 像素，绑训练采样率 | 每基元自适应 $\frac{s}{\hat{\nu}_k^2}$，Nyquist 上界 | 覆盖单像素，近似 box filter |
| 归一化 / 能量修正 | **无**（峰值不变、能量被放大） | **有** | **有** |
| 何时起作用 | 训练 + 渲染（有害） | 烘焙进表示，训练后固定 | 渲染时 |
| 解决的伪影 | —— | 拉近的高频伪影 / 腐蚀 | 拉远的混叠 / 膨胀伪影（变亮） |

**一条主线回顾：** 膨胀在屏幕空间用固定的约 1 像素去补偿"过小高斯"，而"1 像素 = 多大世界空间"随采样率变化 —— 于是训练时凑对的补偿在变焦后失配：拉近时补偿相对不足 → 高斯偏小、违反 Nyquist → 腐蚀 + 高频伪影 / 混叠；拉远时高斯投影小于像素但膨胀不衰减能量 → 错误变亮 + 膨胀伪影。Mip-Splatting 用有采样理论依据（3D 平滑，Nyquist）和有物理依据（2D Mip，成像 box filter + 能量归一化）的两个滤波器替换它，把"频率约束"这件事做对、做在正确的空间里。
